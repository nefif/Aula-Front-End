Sobre esta página

Esta página tem exatamente a mesma estrutura visual da versão feita com tabelas: cabeçalho, menu lateral, conteúdo e rodapé. A diferença é que agora o layout é montado com float, separando estrutura (HTML) de apresentação (CSS).

O menu (<nav>) e o conteúdo (<main>) estão flutuando lado a lado com float: left, cada um com uma largura definida. O rodapé usa clear: both para "descer" abaixo dos elementos flutuantes, já que eles saem do fluxo normal do documento.
