Sobre esta página

Antes do CSS moderno (Flexbox e Grid), era comum usar tabelas HTML para organizar o layout inteiro de um site: cabeçalho, menus, colunas de conteúdo e rodapé eram todos células de uma grande tabela.

Essa técnica funcionava, mas trazia problemas: código difícil de manter, pouca flexibilidade para telas diferentes (nada de responsivo!) e mistura entre estrutura e apresentação.
