🛡️ 6. Travas de Segurança (min-width, max-width, min-height, max-height)

Use propriedades min e max para definir os limites do layout!

1. max-width e min-width

width: 100%

max-width: 500px (Nunca fica gigante no PC)

min-width: 280px (Não quebra em telas mini)

2. min-height

min-height: 200px

Garante que o card mantenha uma altura visual mínima bonita, mesmo com pouquíssimo texto dentro dele!

3. max-height + overflow
