Exemplos de float no CSS

1. float: left

A caixa acima foi "flutuada" para a esquerda. O texto ao redor contorna o elemento pelo lado direito, preenchendo o espaço restante. Esse era o comportamento clássico usado para colocar imagens dentro de parágrafos de texto, como em jornais e revistas.

2. float: right

Aqui a caixa foi flutuada para a direita. O texto contorna o elemento pelo lado esquerdo. É o comportamento espelhado do float: left, muito usado para posicionar imagens ou boxes de destaque à direita do conteúdo principal.

3. float: none

Com float: none (o valor padrão), o elemento não flutua: ele se comporta como um bloco normal, ocupando sua própria linha e empurrando o conteúdo seguinte para baixo, sem nenhum texto contornando ao redor.

4. float: inherit

Elemento pai com float: left:

Este texto está dentro da mesma div que a imagem, logo depois dela no HTML. Como a imagem tem float: inherit e herdou float: left do seu elemento pai, o texto contorna a imagem pela direita — exatamente como aconteceria com float: left aplicado diretamente.

O valor inherit faz o elemento herdar o valor de float do seu elemento pai. Como o <div> pai acima tem float: left, a imagem filha com float: inherit também se comporta como float: left — por isso o texto ao lado dela se comporta do mesmo jeito que nos exemplos anteriores. Esse valor é pouco usado na prática, mas é importante para entender o mecanismo de herança no CSS.
