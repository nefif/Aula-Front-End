Sobre esta página

Esta página tem exatamente a mesma estrutura visual das versões anteriores: cabeçalho, menu lateral, conteúdo e rodapé. A diferença agora é que o menu (<nav>) e o conteúdo (<main>) usam display: inline-block em vez de float para ficarem lado a lado.

Note que o rodapé não precisa de clear: both como na versão com float — isso porque elementos inline-block não saem do fluxo normal do documento, então o <footer> simplesmente aparece depois deles sem precisar de nenhum truque especial.

1. Tags (display: inline)

As tags abaixo usam display: inline. Elas ficam lado a lado naturalmente, mas repare que o padding vertical não empurra as linhas de texto ao redor — ele só existe visualmente, "por cima" do conteúdo próximo.

Cada recurso abaixo usa display: block. Mesmo sendo elementos pequenos, cada um ocupa a linha inteira e empilha verticalmente — impossível colocá-los lado a lado sem mudar o display.
